\newcommand*{\DocTitle}{Template for Software Development Reports}
\newcommand*{\DocAbstract}{This is a LaTex template that can be used mostly for Software Development related reports. It contains table of contents, revision history, header/footer support, a predefined-setup for `lstlisting' code blocks and inline defined commands for different text-styles (e.g. inline code, references to files, paths, functions etc). Users of this template can easily change components and add new commands. This template's document is also providing information on how the template is organized. The current version is using single-page style which is most ideal for digital documents (and not printable ones). The template has been tested with MikTex2.9 and TexMaker 5.0.2 using `XeLaTex'.}
